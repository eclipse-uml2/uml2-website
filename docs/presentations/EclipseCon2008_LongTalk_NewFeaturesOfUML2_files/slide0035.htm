Static Profile Definition: Workflow (New)
1.Create our profile in UML.
2.Apply Ecore profile.
3.Apply <<EPackage>> stereotype to the profile root.
4.Set NsURI.
5.Generate code.
6.Register extension point.
7.Apply profile and stereotypes.
8.Get and set values by calling getters and setters.
